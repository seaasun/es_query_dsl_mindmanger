  轻点鼠标，自动生成Elastic search 查询语句
  
* 根据指引选项，点击鼠标自动生成查询语句
* 启用custom模式，可随意添加语句
* 可本地保存查询语句
* 简单快速自定义指引选项

1. 直接访问：[http://whybangbang.github.io/es_query_dsl_mindmanger/](http://whybangbang.github.io/es_query_dsl_mindmanger/ "Title")
